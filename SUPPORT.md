# Support

## Getting Help

For usage questions, setup issues, or general discussion:

- Read the [README.md](./README.md) first
- Read [README.zh-CN.md](./README.zh-CN.md) if you prefer Simplified Chinese
- Search existing issues before opening a new one
- Open a GitHub issue if the problem looks reproducible or project-specific

## Before Opening an Issue

Please include:
- Your browser and version
- Your operating system
- Steps to reproduce
- Screenshots or recordings for UI problems
- Error messages from the browser console if relevant

Issue templates are available in `.github/ISSUE_TEMPLATE/` for:
- Bug reports
- Feature requests
- Usage questions

## Security Issues

Do not report security issues publicly. Follow the process described in
[SECURITY.md](./SECURITY.md).

## AI-Assisted Project Note

This repository is AI-assisted. If you report a bug in generated code or
documentation, a minimal reproduction is especially helpful.
